# Experiment: Grouped Recents UX

This folder contains an experimental version of the `AgentsResearcher` screen with a grouped Recents table UX:
- User-initiated runs are top-level rows.
- Auto-generated runs are collapsed under each parent run.
- Parent rows show auto-run counts and status summaries.
- Large auto-run sets are progressively revealed with "Show more".

Files included:
- `experiment/src/imports/AgentsResearcher.tsx` (experimental component)
- `experiment/src/imports/svg-2wacbqcxt8.ts`
- `experiment/src/assets/eclipse-full.svg`

Main Playground source (`src/`) is intentionally left unchanged.
